# Stoperica u Flowcode

## Opis
Ovo je digitalna stoperica napravljena u Flowcode-u.  
Omogućava start, stop i reset funkcije.

## Kako koristiti
1. Otvori Flowcode projekat iz foldera `flowcode/stoperica.flowcode`.
2. Pokreni stopericu i koristi funkcije Start / Stop / Reset.
